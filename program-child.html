크레용숲 어린이색채학교

CHILD ART · 4~12세

"우리 아이는 지금 어떤 마음으로 세상을 보고 있을까?"

"말로 설명하지 못한 감정은 어디에 남아 있을까?"

아이들은 감정을 먼저 느끼고, 그 마음은 몸·색·선으로 표현되며 그 다음이 비로소 언어입니다. 크레용숲은 아이의 감각·상징·세계가 자기 속도로 싹틀 자리를 지켜줍니다.

이런 친구에게 맞아요
  • 선·색·형태에 호기심이 많고 손으로 탐색하는 걸 좋아하는 아이
  • 감정표현이 서툴지만, 말 대신 그림에서 마음이 잘 드러나는 아이
  • '잘 그리기'보다 자기 속도로 탐색하는 경험이 필요한 아이
  • 정서 조절·감각 자극·표현력 성장을 균형 있게 경험하고 싶은 아이

Curriculum

1년의 감정예술 테마

같은 대주제라도 아이의 나이에 따라 자연스럽게 표현 깊이가 달라집니다. 감정은 반복되어야 구조가 되고, 구조는 축적되어야 자기세계관이 됩니다.

1학기
3월 · 마음의 의상실
"마음에 오늘 어떤 옷을 입힐까?"
감정 패턴·색의 기분·감정의 층을 '의상'으로 상징화
4월 · 마음 우체국
"속마음을 편지 대신 색으로 보낸다면?"
감정-색 연결 / 심리적 거리 표현 / 감정 카드 주고받기
5월 · 가족 감정 아카이브
"우리 가족의 색은 어떤 빛일까?"
가족 감정 지도 / 가족 상징 제작 / 부모 참여형 워크
6월 · 아트키친
"감정은 섞이고 끓고 식기도 해요."
색 혼합 실험 / 감정의 온도 / 텍스처 레시피
7월 · 감정 테라리움
"오늘의 기분을 유리병에 키워볼까?"
감정을 생태·성장 개념으로 다룸
2학기
8월 · 감정 지도
"내 마음은 지금 어디쯤일까?"
감정 영역 / 감정 거리 / 마음 탐험·표현
9월 · 감정 직물 정원
"감정은 반복될 때 결이 됩니다."
선·패턴 작업 → 자기 성향과 감정 리듬 시각화
10월 · 감정 미술관
"내 감정이 작품이 되는 순간"
감정-색-형태의 고도화 / 자기만의 상징 제작
11월 · 감정 표본 박물관
"1년의 감정을 보존하는 과학적 예술 실험"
감정 포트폴리오 / 감정 표본 채집 / 기억의 색 기록
12월 · 마음 서랍
"마음의 조각들을 한 서랍에 모아두는 시간"
1년의 감정 기록 → 개인 아카이브북 완성
